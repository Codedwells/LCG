## 🥂 After making a change

1. Create a new branch

```bash
git checkout -b YourBranchName
```

2. Add it to staging area

<br>

> NOTE: don't commit the package.json

```bash
git add <path to the file you worked on>
```

3. Commit your changes with

```bash
git cz
```

4. Push your changes

```bash
git push
```
